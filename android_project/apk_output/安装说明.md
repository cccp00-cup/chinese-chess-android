# 中国象棋 - Android安装包说明

## 📱 可用安装包

本目录包含以下Android安装包，支持不同的安装方式：

### 1. Kivy Launcher版本 (ChineseChess_KivyLauncher.zip)
- **文件大小**: ~200KB (压缩包)
- **安装方式**: 通过Kivy Launcher运行
- **适用场景**: 开发测试、快速体验
- **优点**: 无需编译，文件小，更新方便
- **缺点**: 需要额外安装Kivy Launcher

### 2. PyDroid版本 (ChineseChess_PyDroid.zip)
- **文件大小**: ~250KB (压缩包)
- **安装方式**: 通过PyDroid 3运行
- **适用场景**: 个人使用、学习开发
- **优点**: 完整的Python环境，支持pip安装依赖
- **缺点**: 首次运行需要安装依赖

### 3. Web PWA版本 (ChineseChess_Web_PWA.zip)
- **文件大小**: ~180KB (压缩包)
- **安装方式**: 作为渐进式Web应用安装
- **适用场景**: 跨平台使用、无需安装Python环境
- **优点**: 真正的APK体验，支持离线运行
- **缺点**: 功能可能受限

## 🚀 推荐安装方式

### 方法1: PyDroid版本 (推荐)
1. 在Google Play商店搜索并安装 **PyDroid 3**
2. 将 `ChineseChess_PyDroid.zip` 解压到手机存储
3. 在PyDroid中打开解压后的 `install_dependencies.py` 安装依赖
4. 安装完成后运行 `run_game.py` 开始游戏

### 方法2: Kivy Launcher版本
1. 在Google Play商店搜索并安装 **Kivy Launcher**
2. 将 `ChineseChess_KivyLauncher.zip` 解压到手机存储
3. 在Kivy Launcher中选择中国象棋项目
4. 点击运行开始游戏

### 方法3: Web版本 (最简单)
1. 将 `ChineseChess_Web_PWA.zip` 解压到手机存储
2. 用浏览器打开 `index.html`
3. 添加到主屏幕作为Web应用
4. 点击图标开始游戏

## 🎯 功能特色

✅ **完整象棋规则**: 支持所有象棋规则和走法  
✅ **网络对战**: 支持局域网和互联网对战  
✅ **AI对手**: 内置智能AI，支持多种难度  
✅ **音效支持**: 包含完整的游戏音效  
✅ **界面美观**: 传统中国风界面设计  
✅ **跨平台**: 支持Android各种版本  

## 🔧 系统要求

- **操作系统**: Android 5.0 (API 21) 或更高
- **存储空间**: 至少 50MB 可用空间
- **网络**: 网络对战需要网络连接
- **权限**: 网络访问权限（用于联机功能）

## 🎮 快速开始

## 🔧 故障排除

### 游戏无法启动
- 确保已安装正确的运行环境（PyDroid 3 或 Kivy Launcher）
- 检查文件是否完整解压
- 确认手机有足够的存储空间

### 网络连接失败
- 检查网络连接是否正常
- 确认防火墙没有阻止应用
- 尝试重启应用或手机

### 音效无法播放
- 检查手机音量设置
- 确认应用有音频权限
- 尝试重启应用
